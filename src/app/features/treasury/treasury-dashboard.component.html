This dashboard uses the GSA Federal Real Property Profile (FY2023), a public domain dataset covering ~270,000 federal real estate assets, as a proxy for a corporate real estate portfolio. Predictive methodology: 2-period linear trend extrapolation applied to utilization rate (current vs. simulated prior-year). Assets below 60% are flagged under the federal USE IT Act benchmark.
